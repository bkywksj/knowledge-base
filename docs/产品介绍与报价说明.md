# Knowledge Base 智能知识库

> 一款本地优先、AI 驱动的桌面知识管理软件
> 让笔记不只是存下来，而是被真正"想起来、用起来"

---

## 一、为什么需要它（痛点）

笔记越记越多，却越来越找不到、用不上——

- 资料散落在微信收藏、浏览器书签、各种文档里，**想用的时候永远找不到**；
- 记了几百条笔记，却从没回看过，沦为"数字垃圾场"；
- 想问 AI 一个问题，它只会泛泛而谈，**完全不了解你自己积累的资料**；
- 在线笔记软件要联网、要订阅、数据在别人服务器上，**断网即罢工、隐私无保障**。

知识的价值不在于"存了多少"，而在于"能否在需要的那一刻被调用"。

---

## 二、它是什么（解决方案）

**Knowledge Base** 是一款基于 Tauri 构建的原生桌面应用，把"记录—整理—检索—调用"做成一条完整闭环：

你录入的每一条笔记，都会成为**你专属 AI 助手的知识来源**。问它问题，它会基于你自己的笔记作答，并标注引用了哪几条、附上原文图片——这不是通用聊天机器人，而是**懂你资料的第二大脑**。

数据全部存储在本地 SQLite，**断网可用、隐私自留**，同时支持多端同步与移动端访问。

## 四、亮点功能：MCP 双向互联（接入 AI 生态）

支持 **MCP（Model Context Protocol，模型上下文协议）**，让你的知识库不再是一座孤岛，而是融入整个 AI 工作流：

- **作为 MCP 服务对外开放**：一键安装到 **Claude Desktop / Cursor / Claude Code** 等主流 AI 客户端后，这些工具就能直接搜索、读取（乃至创建、更新）你的本地笔记——你积累的资料瞬间变成它们的"长期记忆"；
  - 提供搜索笔记、读取笔记、列出标签/任务/日记/文件夹、回收站还原、创建笔记、加标签等多种工具；
  - **默认只读，更安全**；可显式开启写权限，让 AI 帮你整理与录入。
- **作为 MCP 客户端接入外部服务**：在「设置 → MCP 服务器」中添加任意第三方 MCP 服务，应用内的 AI 即可调用它们，按需扩展能力边界。

> 一句话：**让你的知识，能被你最常用的 AI 工具随时调用；也让外部 AI 能力，反哺你的知识库。**

## 五、它和别的笔记软件有什么不同

- **本地优先**：数据存在你自己电脑上，不强制联网、不依赖云端、不卖你的数据；
- **AI 真正读你的笔记**：基于检索增强（RAG）作答并溯源，而非通用大模型空谈；
- **接入 AI 生态**：通过 MCP 双向打通 Claude / Cursor 等工具，不做信息孤岛；
- **原生轻量**：Tauri 架构，安装包小、启动快、占用低，不是套壳网页；
- **一次付费，长期使用**：买断授权，不做按月订阅的"年费陷阱"。
