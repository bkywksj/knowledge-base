任务管理模块 · UI 原型

v1 设计稿 · Tauri + React + Ant Design + TailwindCSS

1. 列表视图

主入口 · 按时间分组（逾期 / 今天 / 本周 / 已完成）· 紧急度圆点 · 关联 chip · hover 快捷操作

2. 看板视图

按紧急度分三列 · 拖拽改优先级 · 紧凑卡片 · 空列可引导新建

3. 日历视图

月视图 · 事件色带（紧急=红 / 一般=蓝 / 不急=灰 / 已完成=绿删除线）· 底部"未安排日期"抽屉

4. 新建任务 Modal

标题 + 紧急度 + 截止时间（今天/明天/本周末快捷）+ 关联（笔记/目录/链接三选）· Cmd↵ 保存

设计要点回顾

  • 紧急度三色：红 / 蓝 / 灰，对应 urgent / normal / low，一维足够覆盖"紧急任务 vs 不急任务"的诉求
  • 关联多态：笔记（蓝色 chip）/ 目录路径（紫色 chip）/ 外部链接（绿色 chip），点击用 tauri-plugin-opener 跳转
  • 与现有 UI 一致：左侧边栏沿用当前导航结构，"待办"作为一级入口，未完成数显红色徽章
  • 快捷输入：新建标题支持 [[笔记标题]] 自动挂关联；Ctrl/Cmd+Enter 保存；今天/明天/本周末快捷日期
  • 分组而非过滤：列表视图用"逾期/今天/本周"分组标题，比单一筛选器更有信息密度
