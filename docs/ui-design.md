### 侧边栏结构（3段式）

- **导航菜单**：首页 / 笔记 / 搜索 / 每日笔记 / 标签管理 / 关于
- **文件夹树**：可展开/折叠的文件夹结构，右键菜单
- **快捷入口**：回收站

## 实施优先级

| 优先级 | 页面 | Phase |
|--------|------|-------|
| P0 | 首页（快速搜索+最近编辑+统计） | 1 |
| P0 | 笔记列表（列表+卡片视图切换） | 1 |
| P0 | 笔记编辑器（标题+TextArea） | 1 |
| P0 | 侧边栏（3段式：导航+文件夹树+快捷入口） | 1 |
| P1 | 搜索页（全文搜索+筛选） | 2 |
| P1 | 每日笔记（日历+编辑器） | 2 |
| P1 | 标签管理（标签云+关联笔记） | 2 |
| P1 | 回收站（软删除+恢复） | 2 |
| P2 | 设置页（外观+AI+数据） | 3 |
| P2 | AI 对话（RAG 知识问答） | 3 |
| P2 | 知识图谱（@antv/g6 可视化） | 4 |
